Projects

Snapchef

Snapchef is a web application that generates recipes based on the ingredients detected with our AI model using webcam or image input. Once ingredients are determined, ChatGPT API will generate recipes. We used YoloV8, PyTorch, OpenCV, to train our model on 80+ ingredients. We used a with a 95% classification accuracy. Users can see past requests and are authenticated via BCrypt with their data being stored in MongoDB. Our backend was built with both Flask and Node.js.

React Node.js MongoDB PyTorch OpenCV JavaScript Flask

AWS Global Rankings Hackathon

For the AWS Global Rankings Hackathon, we created a RestAPI to rank the international and regional rankings for League of Legend teams. We used XGBoost to predict outcomes of tournaments, conducting feature engineering and analysis on over 1000+ matches. Our API was built with API Gateway, Lambda, S3, and SageMaker.

AWS TypeScript Python Pandas Sci-kit Numpy

Chat-GPTest

I programmed a VSCode extension to use the ChatGPT API to generate unit tests for primarily Python, C++, and JavaScript code. The extension can write unit tests based on a prompt and existing code written by a developer. In addition, it can also debug and provide insights on failing unit tests. The extension uses popular testing frameworks such as PyTest and Jasmine.
